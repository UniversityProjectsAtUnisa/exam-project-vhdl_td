# traccia8

## Scarica il progetto

git clone https://github.com/marco741/traccia8.git

## Caricare versioni modificate del progetto

1. Aggiorna la tue versione della libreria con 
git pull
2. Esegui le modifiche desiderate
3. Aggiungi tutti i file che vuoi caricare alla lista di caricamento con
git add -A
4. Aggiungi un commento significativo delle tue modifiche con
git commit -m "[messaggio]"
5. Carica le modifiche effettuate sulla repository con
git push

## ATTENZIONE
Nessuno dei precedenti passaggi è opzionale: Prima di eseguire modifiche occorre sempre fare git pull e il messaggio è obbligatorio
